⚠️ TROUBLESHOOTING
══════════════════════════════════════════════════════════════════════════════

  Backend not responding?
    → Start: cd server && npm start
    → Check: curl http://localhost:5000/

  Products not loading?
    → Check MongoDB connection
    → Check API returns: curl http://localhost:5000/api/products

  Login fails?
    → User exists in database?
    → Check password (create new account to test)

  Cart empty?
    → Must be logged in (cart requires auth)
    → Check localStorage.authToken exists

  CORS error?
    → Ensure backend has cors() middleware
    → Check server.js app.use(cors())

  See QUICK_START.md for more troubleshooting
